Mega PDF — Privacy Policy

Published by Electric RV · Effective July 22, 2026

The short version: Mega PDF does not collect, transmit, store remotely, or share any personal data or document content. Everything the app does happens on your device.

Your documents

PDF files you open, edit, sign, or print are processed entirely on your device. Mega PDF has no server component, requires no account, and never uploads your documents or any part of them — including file names, text, images, or signatures — anywhere.

Data stored on your device

Mega PDF keeps a small amount of data locally under your Windows user profile (%LOCALAPPDATA%\MegaPDF): application settings, signature images you add, and a crash-recovery journal of unsaved edits. This data never leaves your device and is removed when you uninstall the app or delete that folder.

Network access

The Microsoft Store version of Mega PDF makes no network connections. App updates are delivered by the Microsoft Store itself, under Microsoft's privacy statement.

Preview builds distributed outside the Store check GitHub for updates by querying the public GitHub releases feed. That request contains no personal data or document information, though GitHub receives standard connection metadata such as your IP address, as described in the GitHub privacy statement.

Telemetry and analytics

None. Mega PDF contains no telemetry, analytics, advertising, or tracking of any kind. If a future version ever offers optional diagnostics, it will be off by default, strictly opt-in, will never include document content or file names, and this policy will be updated first.

Children's privacy

Mega PDF collects no data from anyone, including children.
